# Modal

A Modal interrupts the app experience to prompt a user to confirm an action or acknowledge a piece of information. The modal header, content and footer are exposed as slots, allowing control of what goes in them.

## Guidelines

-   [Astro UXDS: Modal](https://astrouxds.com/ui-components/dialog-box)

export const Default = (args) => {
    return html`
        <div style="display: flex; flex-flow: column; justify-content: center;">
            <rux-modal id="default-modal" ?open="${args.open}">
                <h4 slot="header">Modal Title</h4>
                <div slot="content">
                    <rux-input
                        placeholder="Modal Input"
                        label="Modal Content"
                    ></rux-input>
                </div>
                <div
                    slot="footer"
                    style="width: 100%; display: flex; justify-content: space-between; align-items: center;"
                >
                    <div>
                        <a
                            href="#"
                            style="margin-right: 10px; text-decoration: none; color: var(--color-primary);"
                            >Modal Link</a
                        >
                    </div>
                    <div>
                    <rux-button-group>
                        <rux-button id="cancel-btn" secondary>Cancel</rux-button>
                        <rux-button id="confirm-btn">Confirm</rux-button>
                    </div>
                </div>
            </rux-modal>
        </div>
        <script>
        let defaultModal = document.getElementById('default-modal');
        let confirmBtn = document.getElementById('confirm-btn');
        let cancelBtn = document.getElementById('cancel-btn');
        confirmBtn.addEventListener('click', () => {
            defaultModal.open = false;
        })
        cancelBtn.addEventListener('click', () => {
            defaultModal.open = false;
        })
        </script>
    `
}

## Cherry Picking

If you're already utilizing a build system that supports tree shaking and want to only import this individual component:

```js
import RuxModal from '@astrouxds/astro-web-components/dist/components/rux-modal.js'
import RuxButton from '@astrouxds/astro-web-components/dist/components/rux-button.js'
customElements.define('rux-modal', RuxModal)
customElements.define('rux-button', RuxButton)
```
