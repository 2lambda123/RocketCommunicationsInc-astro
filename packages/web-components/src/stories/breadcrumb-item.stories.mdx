# Breadcrumb Item

Breadcrumb Item belongs to the Breadcrumb Component. When used in conjunction with Breadcrumd, each Breadcrumb Item except the last will have a chevron-right separator icon after it for separation.

## Guidelines

-   [Astro UXDS: Breadcrumb](https://www.astrouxds.com/components/breadcrumb/)

export const Default = (args) => {
    return html`
<div style="padding: 5%;">
    <rux-breadcrumb-item href=${args.href} current=${args.current}>Breadcrumb Item</rux-breadcrumb-item>
</div>
    `
}
