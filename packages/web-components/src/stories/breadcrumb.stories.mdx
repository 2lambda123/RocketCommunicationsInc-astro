# Breadcrumb

A Breadcrumb component is a secondary navigation mechanism that can improve users' sense of locational awareness within a site or app's hierarchy.

## Guidelines

-   [Astro UXDS: Breadcrumb](https://www.astrouxds.com/components/breadcrumb/)

export const Default = (args) => {
    return html`
<div style="padding: 5%;">
    <rux-breadcrumb>
        <rux-breadcrumb-item href="#">Home</rux-breadcrumb-item>
        <rux-breadcrumb-item href="#">Second Item</rux-breadcrumb-item>
        <rux-breadcrumb-item>Current Item</rux-breadcrumb-item>
    </rux-breadcrumb>
</div>
    `
}

## API

<ArgsTable of="rux-breadcrumb" />

##### *Note:* Any html element placed into the default slot of rux-breadcrumb will be automatically assigned its own slot. For instance, if three `<rux-breadcrumb-item>`s are assigned to the default slot of `<rux-breadcrumb>`, they will automatically be assigned `slot="slot-1"` `slot="slot-2"` and `slot="slot-3"` respectively.

## Variants

### With Icons
Rux-icons can be added to any breadcrumb item.

export const Icons = (args) => {
    return html`
<div style="padding: 5%;">
    <rux-breadcrumb>
        <rux-breadcrumb-item href="#"><rux-icon icon="home"></rux-icon> Home</rux-breadcrumb-item>
        <rux-breadcrumb-item href="#"><rux-icon icon="border-clear"></rux-icon> Second Item</rux-breadcrumb-item>
        <rux-breadcrumb-item><rux-icon icon="person"></rux-icon>Current Item</rux-breadcrumb-item>
    </rux-breadcrumb>
</div>
    `
}

<Canvas>
    <Story
        name="Icons"
    >
        {Icons.bind()}
    </Story>
</Canvas>

### Icons Only
Breadcrumb items can contain only an icon as well

export const IconsOnly = (args) => {
    return html`
<div style="padding: 5%;">
    <rux-breadcrumb>
        <rux-breadcrumb-item href="#"><rux-icon icon="home"></rux-icon></rux-breadcrumb-item>
        <rux-breadcrumb-item href="#"><rux-icon icon="border-clear"></rux-icon></rux-breadcrumb-item>
        <rux-breadcrumb-item><rux-icon icon="person"></rux-icon></rux-breadcrumb-item>
    </rux-breadcrumb>
</div>
    `
}

## Cherry Picking

If you're already utilizing a build system that supports tree shaking and want to only import this individual component:

```js
import { RuxBreadcrumb } from '@astrouxds/astro-web-components/dist/components/rux-accordion'
import { RuxBreadcrumbItem } from '@astrouxds/astro-web-components/dist/components/rux-accordion-item'
customElements.define('rux-breadcrumb', RuxBreadcrumb)
customElements.define('rux-breadcrumb-item', RuxBreadcrumbItem)
```
