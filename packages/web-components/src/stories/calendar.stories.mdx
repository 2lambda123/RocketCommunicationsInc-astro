## Variants

### With date-in

The `date-in` property controls the default date of the calendar. Use this when you want to render
a calendar at a certain date.

export const DateIn = (args) => {
  return html`
  <div style="padding: 10%; display: flex; justify-content: center;">
    <rux-calendar date-in="${args.dateIn}" min="${args.min}" max="${args.max}"></rux-calendar>
  </div>
  `
}

### With min and max

Setting the min and max controls how far back in time or forward in time the calendar can go.

}
