# Astro Tailwind Theme

> The colors and variables used in this are under heavy active development and will change.


## Installation 

* [Install Tailwind](https://tailwindcss.com/docs/installation) in your project.
* Copy over the `tailwind.config.js` from the [Astro Tailwind repo](https://github.com/RocketCommunicationsInc/astro-tailwind/blob/main/tailwind.config.js) into the root of your project.

## Usage 

For now, reference [tailwind.config.js](https://github.com/RocketCommunicationsInc/astro-tailwind/blob/main/tailwind.config.js) for a list of available colors and utilities.
> These will be available eventually on our Design Token documentation page.
